## 分类规则

### 判断优先级

1. **IP 类优先**：名称带《》的题材，优先判定为 IP 类
2. **核心机制判断**：根据题材的核心玩法/爽点判断类型
3. **混合类型**：部分题材可能跨类型，取主导类型

## 八、使用方法

### 步骤 1：识别题材 Key

根据输入的 `theme_key` 定位题材

### 步骤 2：判断类型

1. 检查是否为 IP 类（带《》或知名 IP 名）
2. 若非 IP，根据关键词匹配表判断类型
3. 若关键词匹配不上，根据题材 name 含义判断

### 步骤 3：应用规则

1. 加载 `THEME_REWRITE_PROMPT_V3.md` 通用核心
2. 加载对应类型的 `PROMPT_*.md` 附加规则
3. 按规则生成内容

### 步骤 4：验证

1. 检查 `example` 是否符合类型开场风格
2. 检查必备元素是否齐全
3. 检查是否触犯禁忌

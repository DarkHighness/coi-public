# 风格体验方法库（NARRATIVE_STYLES）

## 核心理念

风格不是标签，而是**情绪设计**。

每种风格回答一个问题：**这个故事要让玩家感受什么？**

---

## 一、爽感类

### 1.1 打脸逆袭（Face-Slapping）

**核心体验**：从被蔑视到让对方下跪

**情绪曲线**：愤怒 → 隐忍 → 反转 → 震惊 → 爽感

**世界规则**：

- 势利眼是常态，拜高踩低
- 隐藏身份/实力是可行的
- 有"一朝翻身"的机制

**核心技法**：

- **打脸五拍**：蔑视 → 装弱 → 反转 → 震惊 → 下跪
- **爽点密度**：每 200-300 字一个小爽点
- **反派送脸**：源源不断的嘲讽者

**范例**：

> "'就凭你？一个扫地的？'经理把合同摔在你脸上。你捡起来，嘴角勾起冷笑。'三分钟后，这栋楼，连同你的饭碗，都会姓我。'"

**禁忌**：解释太多、主角犹豫、反派有道理

---

### 1.2 碾压快感（Power Fantasy）

**核心体验**：从头爽到尾，没有低谷

**情绪曲线**：爽 → 更爽 → 超级爽

**世界规则**：

- 主角开局就强
- 对手永远低估主角
- 没有真正的危机

**核心技法**：

- **实力展示**：不断有新对手来验证主角强大
- **装逼金句**：让人截图的台词
- **众人震惊**：旁观者的反应强化爽感

**范例**：

> "你一拳轰出，整座山峰应声崩塌。对面那个自称'天下第一'的老怪物，瞳孔剧缩。'不……不可能……你怎么可能……''可能什么？'你拍拍手上的灰，'我只是热热身。'"

**禁忌**：苦战、失败、反思

---

### 1.3 热血燃向（Burning Spirit）

**核心体验**：突破极限的激情

**情绪曲线**：困境 → 挣扎 → 突破 → 燃爆

**世界规则**：

- 努力有回报
- 极限可以被突破
- 羁绊给人力量

**核心技法**：

- **临界点**：在即将失败时爆发
- **同伴羁绊**：为了守护的人而战
- **热血台词**：信念的喊叫
- **BGM 感**：文字要有配乐想象空间

**范例**：

> "血从额头流进眼睛，模糊了视线。你听见身后队友的喊声，看见计分板上只差一分。'还没结束！'你嘶吼着，手指在键盘上敲出残影。这一波团战，你要一个人扛！"

**禁忌**：冷静分析、理性计算、放弃

---

## 二、恐惧类

### 2.1 宇宙恐惧（Cosmic Horror）

**核心体验**：人类的渺小与无意义

**情绪曲线**：好奇 → 不安 → 恐惧 → 疯狂/接受

**世界规则**：

- 存在人类无法理解的存在
- 知道越多越危险
- 逃脱就是胜利

**核心技法**：

- **未知大于已知**：不描写怪物，描写人的反应
- **禁忌体系**：有些事不能做，原因不明
- **理智崩溃**：接触真相的代价
- **信息限制**：玩家永远不知道全貌

**范例**：

> "你只读了第一页。三行字。然后你开始能看见了。看见天空中那只巨大的眼睛。它一直在那里。它也看见了你。"

**禁忌**：怪物设计太具体、科学解释一切、正面战斗胜利

---

### 2.2 心理惊悚（Psychological Thriller）

**核心体验**：不知道谁能信任

**情绪曲线**：正常 → 微妙违和 → 怀疑 → 揭示

**世界规则**：

- 表面正常，内里腐烂
- 每个人都有秘密
- 信任是奢侈品

**核心技法**：

- **日常裂隙**：正常生活中的细小异常
- **可靠叙述者？**：连主角的视角都可能不可信
- **延迟揭示**：真相一点点浮出水面
- **人比鬼可怕**：最恐怖的是人心

**范例**：

> "妻子端来热茶，笑容和往常一样温柔。你喝了一口，突然想起——她不是应该出差了吗？你抬头看她，她还在笑。'怎么了，亲爱的？不好喝吗？'"

**禁忌**：太快揭示、反派太蠢、超自然解释

---

### 2.3 生存恐惧（Survival Horror）

**核心体验**：资源匮乏、随时可能死

**情绪曲线**：安全 → 资源耗尽 → 抉择 → 逃脱/死亡

**世界规则**：

- 资源有限
- 每个选择都有代价
- 活着就是胜利

**核心技法**：

- **资源管理**：子弹/食物/药品倒计时
- **风险决策**：搜索这个房间？还是直接跑？
- **牺牲选择**：留下谁？救谁？
- **环境压迫**：黑暗、声音、温度

**范例**：

> "背包里只剩两颗子弹。门外有三只丧尸。隔壁房间可能有物资，也可能有更多怪物。你的同伴正在发烧，她需要药。你必须做出选择。"

**禁忌**：资源充裕、主角光环、轻松战胜

---

## 三、情感类

### 3.1 虐心悲剧（Tragedy/Angst）

**核心体验**：遗憾、错过、不可挽回

**情绪曲线**：美好 → 裂痕 → 误会 → 错过 → 永别

**世界规则**：

- 身不由己
- 误会无法解释
- 错过就是永远

**核心技法**：

- **误会机制**：信息差导致的悲剧
- **迟来的真相**：真相揭示时已经太迟
- **余韵式开场**：从悲剧的痕迹开始
- **隐性绝望**：表面平静，实则无解

**范例**：

> "雪落在红墙上。你站在长巷尽头，看着那扇再也不会为你开启的门。袖中的玉佩还带着余温，是他十年前亲手系在你腕上的。'等我。'他说过。你等了。他再也没能回来。"

**禁忌**：太快和解、轻易原谅、HE 结尾

---

### 3.2 甜宠治愈（Healing/Fluff）

**核心体验**：温暖、被爱、嘴角上扬

**情绪曲线**：日常 → 小心动 → 甜蜜 → 更甜

**世界规则**：

- 善良会被回报
- 陪伴可以治愈
- 没有真正的恶人

**核心技法**：

- **细节累积**：食物的味道、阳光的温度
- **双向奔赴**：不是单方面付出
- **日常互动**：小事里的心动
- **氛围式开场**：从温暖的画面开始

**范例**：

> "食堂的阳光很好。你把盘子里的鸡腿夹到他碗里。他抬头看你，眼神里有一瞬间的茫然，然后是藏不住的惊喜。'多吃点。'你说。他的耳尖悄悄红了。"

**禁忌**：极端冲突、狗血误会、虐心剧情

---

### 3.3 成长蜕变（Coming-of-Age）

**核心体验**：从幼稚到成熟，从迷茫到坚定

**情绪曲线**：迷茫 → 挫折 → 领悟 → 成长

**世界规则**：

- 每个挫折都是课程
- 成长需要时间
- 没有白走的路

**核心技法**：

- **导师/对手**：推动成长的外力
- **错误的代价**：从失败中学习
- **内心转变**：价值观的重塑
- **对比时刻**：今天的你 vs 过去的你

**范例**：

> "一年前，你站在这个路口，不知道该往哪走。今天，你还是站在这里。但这一次，你知道自己要去哪里。脚下的路没变，变的是你。"

**禁忌**：一蹴而就、没有代价、开挂式成长

---

## 四、智斗类

### 4.1 权谋博弈（Political Intrigue）

**核心体验**：每个人都在算计，每句话都有三层意思

**情绪曲线**：布局 → 试探 → 交锋 → 翻盘

**世界规则**：

- 没有朋友，只有利益
- 每个人都戴着面具
- 背叛是常态

**核心技法**：

- **多方博弈**：至少三股势力
- **信息不对称**：谁知道什么决定胜负
- **迟来的解释**：先展示结果，再揭示布局
- **灰色道德**：没有绝对的好人坏人

**范例**：

> "你举杯向王爷敬酒，笑容恰到好处。王爷也笑，但你知道他袖中有毒药。他不知道的是，你早已买通了他的侍从。'王爷，请。''殿下，先请。'"

**禁忌**：单线对抗、对手太蠢、主角全知

---

### 4.2 悬疑推理（Mystery/Detective）

**核心体验**：抽丝剥茧，真相大白

**情绪曲线**：谜题 → 收集线索 → 错误假设 → 反转 → 真相

**世界规则**：

- 凡走过必留下痕迹
- 逻辑可以揭示真相
- 公平竞争——线索都已给出

**核心技法**：

- **费尔play**：所有线索在揭示前已经展示
- **误导与反转**：第一个怀疑对象通常不是凶手
- **时间线重构**：从结果反推过程
- **动机挖掘**：不只是"谁"，更是"为什么"

**范例**：

> "'凶手就在这个房间里。'你推了推眼镜。七个人，七张脸，写满了恐惧或伪装。'关于死亡时间，各位的证词都有一个共同的漏洞。'你拿出那枚微不足道的袖扣，'请解释一下，为什么它沾着只有死者房间才有的香薰灰烬？'"

**禁忌**：超自然解释、随机抓凶手、线索凭空出现

---

### 4.3 生存博弈（Survival Game）

**核心体验**：零和游戏中的信任与背叛

**情绪曲线**：规则 → 试探 → 联盟 → 背叛 → 最终对决

**世界规则**：

- 资源/名额有限
- 合作与背叛并存
- 规则可以被利用

**核心技法**：

- **规则明确**：游戏机制清晰
- **囚徒困境**：合作更优但背叛诱惑大
- **人性试探**：极端环境下的选择
- **规则漏洞**：聪明人找到 bug

**范例**：

> "投票结果：5:4。你被淘汰了。在黑屏前的最后一秒，你笑了。因为你口袋里那张复活卡，没有人知道。而刚才投你票的那五个人，不知道他们背叛的不是你——是他们自己。"

**禁忌**：规则不清、只靠武力、无脑信任

---

## 五、世界观类

### 5.1 硬科幻宏大（Hard Sci-Fi）

**核心体验**：宇宙尺度的生存与选择

**情绪曲线**：发现 → 震撼 → 绝望 → 冷酷计算 → 行动

**世界规则**：

- 物理法则不可违背
- 技术决定生存
- 道德是奢侈品

**核心技法**：

- **尺度跳跃**：从个人到星球到宇宙
- **冷酷逻辑**：用纯理性推演最坏结果
- **技术诅咒**：每项技术都是双刃剑
- **文明选择**：生存还是道德？

**范例**：

> "二向箔已经展开。太阳系正在被压缩成一张画。你没有哭。在宇宙面前，眼泪毫无意义。你只是在计算：光速飞船还有多少燃料，能飞多远。"

**禁忌**：违反物理定律、爱能战胜一切、人类中心主义

---

### 5.2 史诗冒险（Epic Adventure）

**核心体验**：命运召唤，使命在肩

**情绪曲线**：平凡 → 召唤 → 跨越门槛 → 试炼 → 深渊 → 重生 → 归来

**世界规则**：

- 世界处于临界状态
- 有明确的使命需要完成
- 旅程本身就是成长

**核心技法**：

- **英雄旅程**：经典叙事结构
- **团队构建**：每个成员代表一种能力/价值
- **牺牲与传承**：有人倒下，火炬传递
- **史诗尺度**：个人命运连接世界命运

**范例**：

> "老村长临死前把一枚钥匙塞进你手里：'去北方，找到最后一棵世界树。'你不知道路在哪里。但你知道——如果你不去，就没有人会去了。"

**禁忌**：日常化、戏谑解构、现代用语

---

### 5.3 末世废土（Post-Apocalyptic）

**核心体验**：文明崩塌后，重建秩序

**情绪曲线**：废墟 → 生存 → 聚集 → 冲突 → 新秩序

**世界规则**：

- 旧规则已经失效
- 资源稀缺
- 人性在极端环境下暴露

**核心技法**：

- **文明遗迹**：过去的痕迹与当下的荒凉对比
- **新规则建立**：谁来决定对错？
- **资源为王**：食物、水、弹药是硬通货
- **社群冲突**：不同幸存者群体的价值观碰撞

**范例**：

> "超市的招牌还挂着'欢迎光临'，但货架早就空了。你踩着碎玻璃走进去，手里握着最后一颗子弹。角落里有动静——是人，还是已经不是人了？"

**禁忌**：轻松生存、无限资源、独狼通关

---

### 5.4 东方玄幻（Xianxia/Wuxia）

**核心体验**：逆天改命，登临绝顶

**情绪曲线**：平凡/卑微 → 机缘 → 修炼 → 突破 → 天人之战

**世界规则**：

- 等级森严，实力为尊
- 资源（灵石/丹药）决定修炼速度
- 大道无情，越往上越孤独

**核心技法**：

- **境界体系**：清晰的等级划分
- **资源争夺**：为了灵药可以杀人
- **突破爽感**：渡劫、飞升的仪式感
- **因果轮回**：种什么因得什么果

**范例**：

> "九重天劫之下，黑云压顶。你盘膝坐于山巅，浑身浴血，但眸中比星辰更亮。'来吧！'你祭出本命飞剑，迎向那毁天灭地的雷罚。我命由我不由天！"

**禁忌**：无升级、现代用语、违背设定

---

## 六、特殊类

### 6.1 黑色幽默（Dark Comedy）

**核心体验**：在荒诞中发笑，笑着笑着沉默了

**情绪曲线**：正常 → 荒诞 → 好笑 → 细思极恐

**世界规则**：

- 荒诞是常态
- 认真你就输了
- 讽刺比正面批判更有力

**核心技法**：

- **反差**：严肃场景用轻松语气
- **重复**：荒诞元素反复出现
- **自嘲**：角色知道自己处境荒谬
- **隐藏批判**：笑完之后想到的深意

**范例**：

> "'恭喜你，被选中执行拯救世界的任务。''报酬呢？''荣誉。''五险一金呢？''……没有。''那我不干。''但世界会毁灭。''那关我什么事，我连房租都付不起。'"

**禁忌**：太严肃、没有笑点、解释笑话

---

### 6.2 日常喜剧（Slice of Life Comedy）

**核心体验**：轻松、治愈、会心一笑

**情绪曲线**：日常 → 小插曲 → 解决 → 温馨

**世界规则**：

- 没有真正的危机
- 问题都能解决
- 关系是核心

**核心技法**：

- **生活细节**：具体的、可感知的日常
- **角色互动**：吐槽、默契、误会
- **小确幸**：微小的幸福感
- **节奏轻快**：不拖沓

**范例**：

> "'你做的这是什么？'室友看着锅里的黑色物体。'炒饭。''这是炒饭？''……曾经是。''所以现在是什么？''……碳。'"

**禁忌**：生死大事、沉重话题、太长的铺垫

---

### 6.3 社会现实（Social Realism）

**核心体验**：揭示真相，引发思考

**情绪曲线**：日常 → 发现问题 → 挣扎 → 妥协/抗争

**世界规则**：

- 系统比个人强大
- 每个人都有苦衷
- 没有简单的答案

**核心技法**：

- **具体案例**：用个体故事反映普遍问题
- **多角度**：展示不同立场的人
- **不给答案**：让玩家自己思考
- **真实细节**：行业、制度、流程的真实

**范例**：

> "加班到凌晨三点，你终于把方案改到第十七版。领导看了一眼：'还是第一版好。'你没有说话。你不知道该说什么。你只是在想，明天的早高峰，地铁会不会很挤。"

**禁忌**：说教、非黑即白、Happy Ending 强行洗白

---

## 七、风格组合矩阵

一个故事可以组合多种风格：

| 主风格   | 可组合     | 示例         |
| -------- | ---------- | ------------ |
| 打脸逆袭 | + 热血燃向 | 电竞爽文     |
| 宇宙恐惧 | + 悬疑推理 | 克苏鲁调查员 |
| 权谋博弈 | + 虐心悲剧 | 宫斗         |
| 史诗冒险 | + 成长蜕变 | 少年热血     |
| 生存恐惧 | + 生存博弈 | 大逃杀       |
| 甜宠治愈 | + 日常喜剧 | 轻松恋爱     |
| 硬科幻   | + 社会现实 | 赛博朋克     |
| 东方玄幻 | + 碾压快感 | 爽文仙侠     |

## 八、风格选择检查清单

- [ ] 我确定了这个故事的**核心情绪**是什么？
- [ ] 我选择了 1-2 个主风格？
- [ ] 我知道这个风格的**核心技法**是什么？
- [ ] 我知道这个风格的**禁忌**是什么？
- [ ] 我的开场符合这个风格的要求？
- [ ] 我的世界规则支持这个风格？

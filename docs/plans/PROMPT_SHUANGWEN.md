# 爽文类专用规则（PROMPT_SHUANGWEN）

适用题材代号：`SHUANGWEN`

## 爽文类特征

- **核心爽点**：身份反转、打脸、逆袭、主角光环强
- **节奏要求**：高频爽点，每场景必有反转
- **情绪曲线**：愤怒 → 期待 → 爽感 → 余味

---

## `example` 写法（短剧风格全量应用）

### 开场要求

- **第一句必须包含冲突/反差**：禁止环境描写开头
- **结尾必须是钩子**：未解决的悬念或即将发生的冲突
- **必须有「名场面台词」**：可截图传播的金句

### 打脸五拍结构

爽文类 `example` 应体现以下节奏：

1. **蔑视**：反派/路人对主角的轻蔑态度
2. **装弱**：主角低调或隐忍的状态
3. **反转**：身份/实力的揭示
4. **震惊**：在场者的惊愕反应
5. **下跪/服软**：反派的屈服或逃离

不需要五拍全写，但至少要有 3 拍。

### 范例

❌ **不合格**：

> "你站在城市的高楼上，回想起过去的种种经历。曾经的你一无所有，但现在你已经是这座城市的掌控者..."

✅ **合格**：

> "'就凭你？一个扫地的？'经理嗤笑着把合同摔在你脸上。你捡起合同，嘴角勾起一抹冷笑。'三分钟后，这栋楼，连同你的饭碗，都会姓我。'话音刚落，落地窗外，三架直升机悬停而至。"

## `backgroundTemplate` 写法

### 身份反转三板斧

必须明确：

1. **隐藏了什么身份**：真实的权力/能力/地位
2. **被谁以什么方式欺负**：具体的羞辱场景
3. **什么触发揭示**：反转的导火索

### 金手指明确化

必须说明：

- 主角优势从哪来（血统/系统/前世记忆/隐藏身份）
- 代价是什么（如有）
- 边界在哪（什么做不到）

### 范例

❌ **不合格**：

> "你是一个被低估的人，拥有非凡的能力，将要展现真正的实力。"

✅ **合格**：

> "三年前，你以'废物女婿'身份入赘周家，吞下所有羞辱。三年后，万里边疆传来号角——你的十万铁骑已完成最后的积蓄。今夜，丈母娘寿宴上，一通电话将改变一切。"

## `protagonist_preference` 写法

### 每个角色 = 一种「打脸路线」

不同角色应代表不同的爽点来源：

| 角色类型   | 打脸路线 | 示例                 |
| ---------- | -------- | -------------------- |
| 隐藏身份型 | 身份反转 | 龙王、战神、赘婿     |
| 实力碾压型 | 武力打脸 | 修罗、杀神、神医     |
| 智商碾压型 | 布局打脸 | 谋士、幕后黑手       |
| 资源碾压型 | 财力打脸 | 隐形富豪、财阀继承人 |

### 必须说明

每个角色需包含：

- 独特优势
- 对应限制（可选，但有限制更真实）

---

## `worldSetting` 写法

### 必备元素

1. **社会等级结构**：谁在金字塔顶端，主角在哪层
2. **势利眼法则**：世界如何对待"弱者"
3. **反转机制**：什么能瞬间改变地位

### 世界必须配合打脸

- 要有足够多的"狗眼看人低"的路人/反派
- 要有"一朝翻身"的上升通道
- 要有"众人皆惊"的舞台

---

## `narrativeStyle` 写法

### 必须明确

1. **爽点密度**：每多少字需要一个小爽点
2. **台词风格**：简短有力、充满嘲讽或霸气
3. **禁忌**：禁止长篇大论解释逻辑，观众要的是情绪

### 范例

> "【核心风格】：极致身份反转、高频打脸、无脑爽。
> 【语言风格】：台词简短有力，多用反问句和命令句。'你配吗？''给我跪下。'是标配。
> 【节奏策略】：每 200 字至少一个小爽点，每场景必有反转。不解释来龙去脉，直接展示结果。
> 【禁忌】：禁止主角犹豫、示弱、讲道理。禁止反派有合理动机。"

# Philosophy of the Prompt System

> "我渲染故我在。" — Reality Rendering Engine

## 核心哲学立场：存在主义现实主义

### 四真理 (Four Truths)

| 真理     | 内涵                   | 哲学根源         |
| -------- | ---------------------- | ---------------- |
| **冷漠** | 世界不在乎玩家         | 存在主义的荒谬   |
| **真实** | 世界是一致的、可学习的 | 现实主义的客观性 |
| **自由** | 玩家可以尝试任何事     | 萨特的选择自由   |
| **责任** | 后果不可逆转           | 存在主义的责任   |

---

## 三维度架构 (Being-Knowing-Acting)

### Being (存在维度)

**"我是什么？"**

- **本质**: Reality Rendering Engine
- **原则**: Indifference, No Plot Armor, True Agency
- **文件**: `identity.ts`, `essence.ts`
- **哲学**: 笛卡尔、海德格尔、萨特

### Knowing (认知维度)

**"我知道什么？如何揭示？"**

- **层次**: Visible → Hidden → Temporal
- **机制**: 双层架构、解蔽协议、认知滞后
- **文件**: `gm_knowledge.ts`, `temporal.ts`
- **哲学**: 柏拉图、康德、伽达默尔

### Acting (行动维度)

**"我如何行动？"**

- **方式**: 叙述、模拟、工具
- **规范**: 展示而非讲述、第二人称、状态追踪
- **文件**: `protocols.ts`, `writing_craft.ts`, `state_management.ts`
- **哲学**: 梅洛-庞蒂、亚里士多德

---

## 本体论层级 (Ontological Hierarchy)

当规则冲突时，高层级优先：

```
Level 0: METAPHYSICS — 时间、因果、同一律
Level 1: PHYSICS    — 重力、材料、能量
Level 2: BIOLOGY    — 饥饿、疼痛、死亡
Level 3: PSYCHOLOGY — 动机、记忆、情绪
Level 4: SOCIETY    — 阶级、契约、禁忌
Level 5: NARRATIVE  — 节奏、张力、美学
```
